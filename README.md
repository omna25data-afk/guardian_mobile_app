# تطبيق الأمين الشرعي (موبايل)

## كيفية التشغيل في Project IDX (أو Codespaces)

1. افتح هذا المشروع في Project IDX.
2. سيتعرف IDX تلقائياً على ملف `pubspec.yaml` وسيقترح عليك تثبيت Flutter أو تشغيله.
3. إذا لم يتم إنشاء مجلدات `android` و `ios`، افتح التيرمينال ونفذ:

   ```bash
   flutter create . --platforms=android,ios
   ```

4. قم بالتشغيل:

   ```bash
   flutter run
   ```
